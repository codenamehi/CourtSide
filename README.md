# Hoodlympics Website

## Files in this folder

```
index.html          ← main HTML (upload this)
style.css           ← all styles (upload this)
app.js              ← all JavaScript (upload this)
logo-wordmark.jpeg  ← YOUR LOGO FILE (add this yourself)
```

## How to upload to GitHub

1. Go to your GitHub repo
2. Click "Add file" → "Upload files"
3. Drag ALL 4 files at once (index.html, style.css, app.js, logo-wordmark.jpeg)
4. Click "Commit changes"
5. Your site at https://codenamehi.github.io/CourtSide updates automatically

## Adding your logo

- Name your wordmark logo file exactly: `logo-wordmark.jpeg`
- Put it in the same folder as index.html
- It will appear in the nav and footer automatically
- If the file is missing, the text "HOODLYMPICS" shows instead as a fallback
